# AC Data Mining Project

## Prerequisites

### Install Python

### Clone the repository

```
git clone https://gitlab.up.pt/up202108460/ac-data-mining-project.git
```

### Starting Virtual Enviroment

- Windows:

```
python -m venv venv
.\venv\Scripts\activate
```

- macOS/Linux:

```
python3 -m venv venv
source venv/bin/activate
```

### Install Dependencies:

```
pip install -r requirements.txt
```

### Run Code

```
python3 data_read.py
```

### Stopping Virtual Environment

```
deactivate
```
